## 🛠️ Tech Stack & Skills

### Frontend
- **HTML5 & CSS3:** Semantic structure and advanced styling (Glassmorphism, Flexbox, Grid).
- **JavaScript (Vanilla):** Interactive components, filtering logic, and scroll animations.
- **Styling:** Custom Vanilla CSS with modern typography (Outfit Google Font).
- **Animations:** Intersection Observer for reveal effects and typing animations.

### Backend
- **FastAPI (Python):** High-performance asynchronous backend API.
- **Uvicorn:** ASGI server for running the FastAPI application.
- **Integrations:** EmailJS for contact form handling, Helius API for blockchain data.

### Data & Automation
- **Power BI:** Business Intelligence reporting and data modeling.
- **n8n:** Workflow automation and chatbot design.
- **Python (Pandas/NumPy):** Data cleaning and normalization.
- **Web Scraping:** Automated data extraction systems.

## 🚀 Getting Started

### Prerequisites
- Python 3.10+
- Modern Web Browser

### Local Development

#### 1. Backend Setup
```bash
cd Backend
pip install -r requirements.txt
python main.py
```

#### 2. Frontend Setup
Simply open `Frontend/index.html` in your browser or use a Live Server extension in VS Code.
